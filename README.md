# 📝 Task Tracker Full-Stack App

Простая, чисто для обучения простенькая.

## 🚀 Технологии (Стек)
* **Фронтенд:** Next.js (App Router), React, TypeScript, Tailwind CSS
* **Бэкенд:** C#, ASP.NET Core Web API, Entity Framework Core 
* **База данных:** PostgreSQL
* **Инфраструктура:** Docker, Docker Compose

## 🛠️ Как запустить проект локально

Проект полностью контейнеризирован. Для запуска вам понадобится только установленный **Docker Desktop**.
